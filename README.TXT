## 🚀 Descripción
Asistente especializado en el sector jurídico capaz de procesar texto, imágenes y documentos complejos. Diseñado para optimizar la búsqueda de jurisprudencia, resumen de expedientes y análisis de riesgos legales.

## ✨ Características Clave
* 🧠 **Análisis Multimodal**: Procesa PDFs, imágenes de documentos y texto plano.
* 🔍 **Búsqueda Semántica**: Encuentra conceptos legales, no solo palabras clave.
* 📄 **Generación de Resúmenes**: Sintetiza expedientes extensos en puntos clave.
* 🔒 **Privacidad**: Arquitectura diseñada para el manejo seguro de datos sensibles.

## 🛠️ Stack Tecnológico
* **Core:** Python, LangChain / LlamaIndex.
* **Modelos:** OpenAI GPT-4o / Claude 3.5 Sonnet / Gemini Pro.
* **Vectores:** Pinecone o ChromaDB.
* **Interfaz:** Streamlit o FastAPI.
